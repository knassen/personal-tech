 Specific Statistical Programs
  • Autobox Home Page
    Autobox is a product of Automatic Forecasting Systems, Inc. which can be used to identify and build Box-Jenkins ARIMA and Transfer function models. Models may optionally be selected automatically. Includes graphical analysis tools, automatic intervention detection, and many other features. [for Windows; commercial product]
  • XlispStat Home Page
    Lisp-Stat is an extensible statistical computing environment for data analysis, statistical instruction and research, with an emphasis on providing a framework for exploring the use of dynamic graphical methods. XLISP-STAT is a version of Lisp-Stat based on a dialect of Lisp called XLISP.
    • StatLib Xlispstat
      Boxplot Matrix, Bayesian sensitivity analysis, regression diagnostics using dynamic graphics, routines for the interactive, dynamic, exploratory analysis of survival data, functions and prototypes for fitting generalized linear models, simulate some Markov Chains, functions for the spectral analysis of time series, etc.
    • Projects in LispStat
      Links to projects that use LispStat.
    • Annotated Source Index (LispStat)
      Modules for Lisp-Stat.
    • ViSta Visual Statistics System
      "ViSta is designed for an audience of users having a very wide range of data analysis sophistication, ranging from novice to expert. ViSta provides seamlessly integrated data analysis environments specifically tailored to the user's level of expertise."
    • ViSta Documentation
  • LOCFIT Local Regression & Likelihood
    Available for Unix and Windows 95/NT.
  • Power Analysis Software for PCs
    "A comprehensive list of power analysis software for microcomputers."
  • Mx Homepage
    "Mx is a matrix algebra interpreter and numerical optimizer for structural equation modeling and other types of statistical modeling of data."
  • Mx Examples
    Examples of using Mx for nonlinear, multi-trait multi-method, missing data, and QTL analysis, plus others.
  • Multivariate Data Analysis Software
    "This is a collection of stand-alone routines, in Fortran (mostly) and C. Sample data sets are available. Indications are given on how to compile, link and run. Download the programs and run on your system."
  • The Estima Home Page
    "RATS (Regression Analysis of Time Series) is a leading econometrics/time-series analysis software package. It is used worldwide by economists and others for analyzing time series and cross sectional data, developing and estimating econometric models, forecasting, and much more." [commercial product]
  • The Ethnograph Qualitative Research & Data Analysis Software
    Text-based qualitative data analysis software [commercial product].
  • LIMDEP Information
    "A general econometrics program for estimating linear and nonlinear models, limited and qualitative dependent variable models, models for cross section, time series, and panel data." [commercial product]
  • SAS Institute
    The SAS Institute Home Page. "The SAS System is an integrated suite of software for enterprise-wide information delivery. The functionality of the system is built around the four data-driven tasks common to virtually any application -- data access, data management, data analysis and data presentation." [commercial products]
  • SHAZAM
    "SHAZAM is a comprehensive computer program for econometricians, statisticians, biometricians, sociometricians, psychometricians, politicometricians and others who use statistical techniques.... The primary strength of SHAZAM is for the estimation and testing of many types of regression models." [commercial product]
  • FAQ: comp.soft-sys.shazam
    Frequently asked questions (and answers) about SHAZAM.
  • SPSS Inc.
    Publishers of the Statistical Package for the Social Sciences (SPSS), BMDP Classic for DOS, SYSTAT, and Surveycraft packages. [commercial products]
  • Numerical Recipes Home Page
    Publishers of the Numerical Recipes series of books and software.
  • GAUSS Resources & Archive at American University
    Includes GAUSS code for public, non-commerical use in the areas of Bayesian Econometrics, Covariance Matrix Estimators, DYNARE, Generalized Method of Moments, Goodness of Fit, Multivariate Statistics, Optimization Routines, PDFs and RNGs, Qualitative Choice Models, Regression, Simulation, Simultaneous Equations Estimation, Time Series, Unit Roots and Cointegration, and Utilities
  • |STAT Data Analysis Package
    "|STAT is a set of about 30 data manipulation and analysis programs developed by Gary Perlman at the University of California, San Diego and at the Wang Institute. The manipulation programs are general utilities that work with other standard programs like sort. The analysis programs compute most widely used statistics." For Unix and DOS.
  • MacANOVA Homepage
    "MacAnova is a free, noncommercial, interactive statistical analysis program for Macintosh, DOS/Windows (Win32s), and Unix written by Gary W. Oehlert and Christopher Bingham, both of the School of Statistics, University of Minnesota. MacAnova's strengths are analysis of variance and related models, matrix algebra, time series, and (to a lesser extent) uni- and multi-variate exploratory statistics."
  • ALSCAL Multidimensional Scaling Program
    "ALSCAL performs metric or nonmetric Multidimensional Scaling and Unfolding with individual differences options. It can analyze one or more matrices of dissimilarity or similarity data." Executable for PCs plus Fortran source for other systems.
  • TK Solver Home Page
    "TK Solver is a mathematical modeling platform that simplifies math models into the form of systems of equations and relationships."
  • TK Solver Model Collection
    TK Solver models and tutorials at the Royal Military College of Canada.

 On-Line Statistical Processing
  • General Social Survey
    On-line subsetting and statistics (descriptives, frequencies, crosstabs, weighted, unweighted), codebook search, etc.
  • American National Election Study
    On-line subsetting & statistics, searching, etc.
  • NetMul: WWW Online Multivariate Analysis System (unreachable?)
    "NetMul is an online multivariate analysis system. It provides a Web interface to the ADE-4 multivariate analysis software, which allows [you] to use it from Mac, PC and Unix workstations...To use it, you must first download your data set to our anonymous FTP server..."
  • Globally Accessible Statistical Procedures (G.A.S.P.)
    Users need only choose the routine of interest and then enter the data into a form. It's that simple. Results will be returned to the user's browser window. Includes kernel density estimation, exact power of Fisher exact test, exact, unconditional tests of homogeneity (binomial model) or independence (multinomial model) for 2X2 tables, regression, and others. Some procedures require an Java-enabled browser.
  • UCLA Power Calculator
    Calculate power for a given sample size and sample size for a given power in a number of different contexts.
  • WebStat
    WebStat is a Java application that provides basic statistics such as regression, comparison of means, and summary statistics, as well as several types of plots, via the WWW. Requires a Java-capable broswer.
