FIGlet Fonts & Sites

FIGlet is a computer program for making large letters out of ordinary ASCII text characters. FIGlet lettering is used for web pages, email signatures, text banners, etc.

Kent's Figlet Conversions/Designs
These are my designs and conversions of other people's designs that I've contributed to the FIGlet archives. There are 19 fonts.
More FIGlet Fonts
Explore a large collection of default FIGfonts and contributed fonts from grex.org's figlet fonts directory. I've made a few corrections here and there, or added some characters at the request of grex users. Previews are available on the page.
FIGlet Home Page (Ian Chai)
Answers the questions "What is FIGlet?" "Why does FIGlet exist?" and "Where do I get FIGlet?" This is the main site for all FIGlet activities, including standard and contributed fonts, as well as binaries for several platforms. Ian is the "I" in FIGlet.
An implementation of Figlet in JavaScript
Has a number of effects, including kerning, horizontal and vertical layout.
FIGlet FAQ
This is an old abbreviated FAQ that was posted to the FIGlet mail list. It tells how to subscribe and unsubscribe from the mail list, how to get more fonts, as well as how to reach the main FIGlet site.
Ascii Set FIGlet Server
A web page that lets you pick a figlet font and generate text from it.
Figlet How To and Examples
A short intro to using figlet, including figlet options, and font examples.
Claudio's Figlet Github Tree
Fonts, figlet source code with updates to version 2.2.4. Adds FIGLET_FONTDIR and fixes some code issues.
alt.ascii-art
The Usenet group rec.arts.ascii appears to have gone away when its moderator retired, so alt.ascii-art is the remaining main ASCII art news group. Occasionally, people will post fonts there or announce new FIGlet on-line services and software ports, but beware, without moderation, there is a lot of garbage to sort through. Note: as of 2025 getting usenet is still possible, but not as easy as it was 25 years ago. If you have access to Usenet from your ISP and they get alt.ascii-art, take a look.
Wicked ASCII Art
A very large ASCII art site. This site has not been updated since 1999, so is a static representation of the original site.
bdf2flf in C
Github site for a C language version of bdf2flf utility. It converts a BDF bitmap font to .flf Figlet font format.
bdf2flf in Perl
Github site for Perl language version of bdf2flf utility for converting BDF bitmap fonts to .flf Figlet fonts.
