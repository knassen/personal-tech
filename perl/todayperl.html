today.pl -- Highlight Today's Date in Unix Calendar Output

today.pl is a simple way to highlight today's data in unix 'cal output. Note that more modern cal programs, such as the on from the GNU project do have an option to highlight today's date in the calendar output.

Usage

The command line is: today.pl There are no command line options.
