splitit.pl -- Reformat Multi-column Variable Statements

This Perl script reads a file containing multi-column SAS Input or SPSS Data List statements and reformats them into a single-column format with one field per line. Handy for checking your input declarations when preparing your code or for operating on the statements with scripts.

If all you have is 9 variables to format, by all means, use your editor to do the reformatting, but for larger lists of variables, this program will do the work in a few seconds. That's what I used to use it for, formatting SAS and SPSS statements for large social sciences studies with hundreds or thousands of variables. It was a huge time-saver for me.

  • Output is sent to STDOUT. To save it, redirect STDOUT to a file.
  • Output is one field per line allowing easy review, scripting, and duplicates removal.
  • Developed for quick formatting of SAS/SPSS variable lists. See also wrapit which creates a multi-column format from the output of splitit. splitit and wrapit do the opposite operations, so with them you can go from one command per line to 5 commands per line and back again.

Usage

The command line is: perl splitit.pl filename >filename.out

Here is an example. Consider this input data: 

Sample input (cards):

   V1 1-4         V2 5-8         V3 9-9         V4 10-13 (A)   V5 14-14
   V6 15-16       V7 17-17       V8 18-19       V9 20-21

The command 
perl splitit.pl cards > cards.split produces this:

Sample output (cards.split)

   V1 1-4         
   V2 5-8         
   V3 9-9         
   V4 10-13 (A)   
   V5 14-14 
   V6 15-16       
   V7 17-17       
   V8 18-19       
   V9 20-21
