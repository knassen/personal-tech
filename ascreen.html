Aurora Editor Screen Shots

These are screen shots of my "version" of Aurora: how I've got Aurora/DOS set up in terms of colors, title bars, etc. This is not how Aurora comes "out of the box." However, all the changes I've made to screen appearance were made using Aurora's configuration menu and a little editing in the configuration files. Note that since I generally run my Aurora windows at full screen, I don't like borders on my windows (Aurora provides a simple command to circulate between open windows, even if they aren't visible on the screen); I apologize to those who expect clearly-defined boundaries for their windows...
